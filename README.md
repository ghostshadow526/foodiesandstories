## Commands to push to your repository

1.  **`git init`**: Initializes a new Git repository in your project folder.
2.  **`git add .`**: Stages all the files in your project for the first commit.
3.  **`git commit -m "Initial commit"`**: Saves the staged files as your first commit.
4.  **`git branch -M main`**: Renames the default branch to `main`, which is the standard now.
5.  **`git remote add origin https://github.com/ghostshadow526/icestores.git`**: Connects your local repository to the remote one on GitHub.
6.  **`git push -u origin main`**: Pushes your code to your GitHub repository.
